Dokumentasi Pengguna

Panduan lengkap untuk menggunakan CompuStore. Pelajari cara mengelola produk, melakukan transaksi, dan mengintegrasikan API kami ke dalam aplikasi Anda.

Tentang Platform

CompuStore dibangun dengan teknologi Node.js dan Express untuk memberikan performa tinggi dan skalabilitas. Platform ini dirancang untuk memfasilitasi transaksi jual-beli perangkat komputer dengan aman.

Alur Penggunaan

Ikuti langkah-langkah berikut untuk memulai transaksi di CompuStore:

1
Registrasi

Buat akun sebagai User atau Admin.

2
Eksplorasi

Cari produk dan masukkan keranjang.

3
Checkout

Isi alamat dan metode pembayaran.

4
Tracking

Pantau status pesanan di dashboard.

Hak Akses (Roles)

Administrator
  • Manajemen Produk (CRUD)
  • Manajemen User
  • Update Status Pesanan
  • Akses Laporan Penjualan
Regular User
  • Browsing Katalog Produk
  • Fitur Keranjang & Checkout
  • Melihat Riwayat Transaksi
  • Generate API Key Pribadi
